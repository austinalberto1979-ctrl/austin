# AUSTIN Cosméticos — Site para GitHub Pages

Estrutura correta para publicar:

```
austin/
├── index.html
├── 404.html
└── imagens/
```

No GitHub Pages selecione: **Deploy from a branch → main → /root**.

As avaliações adicionadas no site ficam guardadas no navegador do visitante via localStorage. Para comentários públicos para todos os clientes, ligue uma base de dados como Supabase ou Firebase.
